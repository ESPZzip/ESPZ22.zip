FAQ

¿Es oficial?

No. Este template es para proyectos comunitarios no oficiales. Respeta marcas y derechos de autor.

¿Puedo usar assets de terceros?

Solo si tienes permiso explícito y cumples con sus licencias. Recomendamos gráficos propios o de dominio público.

¿Cómo publico en GitHub Pages?

Crea un repositorio, sube este index.html, activa Pages en Settings → Pages y selecciona la rama principal.

¿Puedo personalizar colores y textos?

Sí. Ajusta las variables CSS en :root y edita el contenido como quieras.
